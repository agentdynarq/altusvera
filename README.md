# Altus Vera

Marketing site for **Altus Vera** — AI-enabled technology solutions, where industry excellence meets academic rigor.

## About this build

This is the production build of the React + Vite site (the canonical source lives in the local `altusvera-react/` repository; see its git history). Built with:

```
vite build --base=/agentdynarq/altusvera/main/
```

Served via raw.githack from this repository's `main` branch. Do not enable GitHub Pages on this account.

The previous hand-written static site (v10) is preserved in this repository's git history prior to the 2026-07-14 deploy.
