# Altus Vera

Marketing site for **Altus Vera** — AI-enabled technology solutions, where industry excellence meets academic rigor.

## About this build

This is an early, **static HTML proof-of-concept** — built with plain HTML, CSS, and vanilla JavaScript, no framework and no build step. The goal here is not the final production site; it is to communicate the **idea, the story, and the overall flow** of the experience before rebuilding it properly in React (or a similar framework).

**It is a work in progress — still mid-build.** Some content is placeholder (for example, the testimonials and partner names are illustrative and not yet confirmed), and several details will change. Treat it as a direction and a storyboard, not a finished product.

## Live preview

Served straight from this repo via a CDN (no build required):

**https://raw.githack.com/agentdynarq/altusvera/main/index.html**

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```

## Structure

- `index.html` — the page
- `css/style.css` — design system and styles
- `js/script.js` — scroll reveals, parallax, engagement path, FAQ
- `images/`, `fonts/` — assets
